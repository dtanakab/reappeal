# README

## サービスページ
https://self-introduction-slideshow.herokuapp.com/

## 本サービスについて

本サービスは、項目にワードを入力するだけで、

簡単に自己紹介スライドショーが作成できるサービスです。
  
作成したスライドのIDとパスワードを共有すれば、

他の人と異なるブラウザ、デバイスでスライドページを共有しながら閲覧することが可能です。
  
## Lint
rubocopとslim-lintを実行  
`$ ./bin/lint`

## Test実行
RSpecテストが実行されます  
  `$ ./bin/test`
